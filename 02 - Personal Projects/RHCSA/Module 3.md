## Quiz 1 Notes

Which directory contains non-persistent process runtime data?

- /tmp
- /etc
- /run
- /var

*Answer: /run* 

Which directory contains installed software programs and libraries?

- /etc
- /lib
- /usr
- /var

*Answer: /usr*
